Best Practices

Choosing the Right Error Correction Level

Error correction determines how much of the QR code can be obscured or damaged while still scanning correctly. Higher levels add redundancy but increase code size.

Level Recovery Capacity Best Used For With Logo? Code Size
L – Low ~7% data recovery Clean digital displays, screens, and apps where damage is unlikely ❌ Not recommended Smallest
M – Medium ~15% data recovery Standard print use — brochures, business cards, packaging ⚠️ Risky Small
Q – Quartile ~25% data recovery Industrial labels, outdoor signage, environments with light damage risk ⚠️ Possible (small logo) Medium
H – High ~30% data recovery Logo overlays, stickers, textured surfaces, QR codes on merchandise ✅ Recommended Largest
Print & Design Tips

Tips for Scannable, Print-Ready QR Codes

A few deliberate choices separate a QR code that scans reliably from one that frustrates users.

SettingRecommendationImportanceNotes
Contrast Dark foreground on light background only 🟢 Critical Inverted (light on dark) QR codes often fail. Never use similar shades for both colors.
Quiet Zone Keep at least 1–2 module border around the code 🟢 Critical Scanners need blank space around the QR pattern to locate the finder squares.
Minimum Size At least 2 cm × 2 cm (0.8") for print 🟡 Important Scanning distance ≈ 10× the code width. At 3 cm, readable from ~30 cm away.
Logo Size Keep logo to 20–25% of QR code area maximum 🟡 Important Too large a logo exceeds even H-level correction and causes scan failures.
File Format SVG for professional print, PNG for web and labels 🟡 Important SVG scales to any size without pixelation. PNG is simpler for most non-design workflows.
Data Length Keep URLs short — use a URL shortener if needed 🟠 Recommended Longer data = more modules = denser code = harder to scan at small sizes.
FAQ

Frequently Asked Questions About QR Codes

Answers to the most common questions about generating and using QR codes.

What is a QR code?

A QR code (Quick Response code) is a two-dimensional matrix barcode that stores data as a grid of black and white squares. Smartphones can read them instantly using the camera app — no dedicated scanner app required on modern devices. They can encode URLs, contacts, WiFi credentials, plain text, and much more.

How do I scan a QR code?

Open your phone's default camera app (works on iPhone and most Android phones) and point it at the QR code. A notification will appear with the encoded action — tap it to open a URL, save a contact, connect to WiFi, and so on. On older devices, use a free QR scanner app like Google Lens.

Can I add a logo to a QR code?

Yes. QR codes with error correction level H (30%) can have up to 30% of their pattern covered without losing readability. Upload your logo using the Logo overlay option, and always set error correction to H when using a logo. Keep the logo to roughly 20–25% of the total QR code area for reliable scanning.

How do I make a WiFi QR code?

Select the WiFi tab, enter your network name (SSID), password, and security type (WPA/WPA2 for most modern routers). Click Generate. When a guest scans the code with their phone, they are automatically prompted to join the network — no password typing required. This works on iOS 11+ and most modern Android devices.

Are my data and passwords stored on a server?

No. All QR code generation happens entirely inside your browser using JavaScript. None of your data — URLs, WiFi passwords, contact details, email content, or any other input — is sent to or stored on any server. Everything remains completely private on your device.

What error correction level should I use?

Use M (15%) for most standard printing on business cards, brochures, and packaging. Use H (30%) whenever you add a logo overlay or when the code will be printed on textured, curved, or potentially damaged surfaces. Use L (7%) only for clean digital displays where the code will never be physically damaged and you want maximum data density.

What is a vCard QR code?

A vCard QR code encodes a digital contact card (vCard format) containing a person's name, phone number, email address, company, job title, website, and postal address. When scanned, the recipient's phone prompts them to save the contact directly to their address book — eliminating manual entry entirely. Ideal for business cards and networking events.

What size should a QR code be for printing?

The minimum recommended size for printed QR codes is 2 cm × 2 cm (approximately 0.8 × 0.8 inches). For posters, the rule of thumb is: scanning distance equals roughly 10 times the code width. A 5 cm code is readable from 50 cm; a 20 cm code from 2 meters. Always use SVG format for print to ensure the code remains crisp at any scale.

What is the difference between static and dynamic QR codes?

A static QR code encodes data permanently in the pattern itself — to change the destination, you must generate and reprint a new code. A dynamic QR code encodes a short redirect URL that can be updated in a dashboard without reprinting. This tool generates static QR codes, which are free forever, require no subscription, and have no expiry date.

Do colored QR codes scan reliably?

Yes, provided there is sufficient contrast between the foreground (dark modules) and background (light modules). The foreground must always be significantly darker than the background. Avoid low-contrast combinations such as dark red on dark blue. Inverted QR codes (light bars on dark backgrounds) are not reliably supported by all scanners and are best avoided.
