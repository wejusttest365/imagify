Understanding Results

What Each Status Code Means

Not every failure is the same — here's how to read the codes the scanner reports.

StatusMeaningTypical Fix
404 The page or file no longer exists at that address Update the link, restore the page, or add a redirect
500 / 503 The destination server errored out or is temporarily down Re-check later; contact the site owner if it persists
3xx Loop The link redirects too many times or into another broken page Point the link directly at the final, working destination
Failed The domain couldn't be resolved, timed out, or refused the connection Verify the domain still exists and spelling is correct
FAQ

Frequently Asked Questions About the Link Checker

Everything you need to know about scanning a site for broken links.

How does the broken link checker scan a whole website?

Enter your homepage URL and start the scan. The tool fetches that page, reads every link on it, then follows internal links to other pages on the same domain, repeating until it reaches your page limit or runs out of new pages to visit. Every link found along the way is checked and reported.

What counts as a broken link?

A broken link is any URL that returns an HTTP error status such as 404 Not Found or 500 Server Error, redirects into an error page, times out, or fails to resolve at all. The tool lists the exact status code or failure reason next to each broken URL.

Can it check external links too, not just my own site?

Yes. Turn on "Also check external links" and the crawler will still only follow and crawl pages on your own domain, but it will test every outbound link found on those pages and report the ones that fail.

Why does scanning rely on a proxy instead of running fully offline?

Browsers block a page from directly reading another domain's raw HTML for security reasons (CORS). To read pages on the site being scanned, the tool routes requests through a public read-only proxy so it can retrieve status codes and page content — nothing is stored, it only passes through to check each link.

Can I export the list of broken links?

Yes. Once a scan finishes, click Export CSV to download the full list of broken links with the page they were found on, the destination URL, and the status code, ready to hand to a developer or import into a spreadsheet.

Does a large site take a long time to scan?

Scan time depends on page count, link volume and the speed setting. A 100-200 page site with heavy external linking can take several minutes; you can stop the scan at any time and still export whatever was found so far.
