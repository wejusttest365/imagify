Tips for Best Results

How to Get the Best Sitemap Results

A few simple choices make a big difference in how search engines treat your sitemap.

Setting Recommendation Benefit Notes
Priority Use "Auto" for large sites, manual for key pages 🟢 Very Important Auto sets priority by URL depth — homepage scores 1.0
Change Frequency Match how often that page's content actually changes 🟡 Important Inaccurate "Always" signals are usually ignored by crawlers
Last Modified Use accurate ISO 8601 dates (YYYY-MM-DD) 🟢 Very Important Helps search engines prioritize re-crawling updated pages
URL Count Keep under 50,000 URLs per sitemap file 🟠 Recommended Larger lists are auto-split into a sitemap index for you
Domain Consistency Only include URLs from one domain or subdomain 🟡 Important Mixed-domain sitemaps may be rejected by search engines
Canonical URLs Only add the canonical version of each page 🟡 Medium Avoid duplicate, tracking-parameter, or redirected URLs
FAQ

Frequently Asked Questions About XML Sitemaps

Everything you need to know about generating a sitemap with WebToolOcean.

What is an XML sitemap and why do I need one?

An XML sitemap is a file that lists your website's pages so search engines can discover and crawl them more efficiently. It's especially useful for new sites, large sites, or pages that are hard to find through internal links alone.

Are my URLs uploaded to a server?

No. The sitemap is built entirely inside your browser using JavaScript. Your URL list is never sent to or stored on any server.

How many URLs can one sitemap contain?

The sitemap protocol limits a single file to 50,000 URLs or 50MB uncompressed. If your list goes over that, this tool automatically splits it into multiple sitemap files plus a sitemap index file that references them.

What's the difference between priority and change frequency?

Priority (0.0–1.0) signals how important a page is relative to other pages on your site. Change frequency hints how often the content typically updates. Both are advisory — search engines may use or ignore them.

Can I import URLs from a file?

Yes. Use "Import from .txt / .csv" to load a list of URLs, one per line (or first column for CSV), directly into the tool.

Where do I upload the sitemap.xml file?

Upload it to your website's root directory, e.g. https://yoursite.com/sitemap.xml, then submit that URL in Google Search Console and Bing Webmaster Tools.

Should I reference my sitemap in robots.txt?

Yes, it's recommended. Add a line like "Sitemap: https://yoursite.com/sitemap.xml" to your robots.txt. Use the "Get robots.txt Line" button to generate it automatically.

Does this tool crawl my website automatically?

No. Browser security restrictions (CORS) prevent a webpage from freely fetching pages from other domains, so this tool keeps everything reliable by letting you add URLs manually, in bulk, or by file import instead of attempting an automatic crawl.

Will adding a sitemap guarantee my pages get indexed?

No. A sitemap helps search engines discover and crawl your pages, but it doesn't guarantee indexing or affect rankings on its own.
